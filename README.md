# TMC2226_StepperDriver_-NK243-01AT-RB
Rotating a motorbank stepper motor model NK243-01AT-RB 10 secs CW and CCW using TMC2226 silent drive with arduinor.

Motor Wire Colors:
White = 2B = +B, Black = 2A=-B,green = 1A = A+, yellow = 1B = -A
